# دراستي – محاكي توجيهي 2026

هذا المشروع عبارة عن موقع ويب بسيط لمحاكاة امتحانات التوجيهي لسنة 2026 باللغة العربية. يوفّر واجهة مرنة تعمل على الكمبيوتر والهواتف الذكية (أندرويد وآيفون) عبر المتصفح، بالإضافة إلى لوح إدارة يسمح برفع الامتحانات بصيغة JSON وعرض نتائج الطلاب. يعتمد المشروع على Google Apps Script لتوفير واجهة API والتخزين في Google Sheets وGoogle Drive.

## مكونات المشروع

```
project/
├── index.html               – واجهة المستخدم التي تُرفع إلى GitHub Pages أو أي خادم ويب
├── README.md                – هذا الملف
└── apps-script/
    ├── Code.gs              – الكود الخاص بGoogle Apps Script
    └── appsscript.json      – ملف إعدادات Apps Script
```

## كيفية الإعداد

1. **إنشاء قاعدة البيانات**:
   - افتح ملفًا جديدًا في Google Sheets (مثلاً باسم `Tawjihi2026DB`).
   - من القائمة، اختر **Extensions → Apps Script**.
   - احذف أي ملفات موجودة وأضف ملفًا جديدًا باسم `Code.gs`، ثم الصق محتويات `apps-script/Code.gs` الموجودة في هذا المشروع.
   - من رمز القائمة للملفات في Apps Script أضف ملف إعدادات `appsscript.json` ثم الصق محتوياته الموجودة في `apps-script/appsscript.json`.

2. **نشر الـ API**:
   - من داخل محرر Apps Script اختر **Deploy → New deployment**.
   - اختر نوع النشر Web app.
   - **Execute as**: اختر `Me`.
   - **Who has access**: اختر `Anyone`.
   - انشر واحصل على رابط الـ Web App.

3. **إعداد واجهة المستخدم**:
   - افتح ملف `project/index.html` في محرر نصوص.
   - غيّر قيمة `API_BASE` في كائن `CONFIG` إلى الرابط الذي حصلت عليه من نشر Apps Script.
   - يمكنك تعديل بقية الإعدادات (مثل كلمة مرور الأدمن أو اسم التطبيق) إذا أردت.

4. **النشر على GitHub Pages أو أي خادم**:
   - ارفع ملف `index.html` إلى مستودع GitHub في الفرع الرئيسي (أو مجلد `docs` حسب تفضيلاتك).
   - من إعدادات المستودع اختر **Pages → Deploy from branch** ثم اختر الفرع والمجلد المناسبين.
   - بعد النشر يمكنك زيارة الرابط وسيعمل الموقع مباشرة.

5. **رفع امتحان جديد**:
   - قم بتسجيل الدخول باسم الطالب من الصفحة الرئيسية.
   - لرفع امتحان جديد انقر على زر **Admin** في أعلى الصفحة وأدخل كلمة المرور الافتراضية (`admin`).
   - داخل لوحة الإدارة الصق نص الـJSON الخاص بالامتحان واضغط على زر **رفع الامتحان**. سيُحفظ الامتحان في Google Drive داخل مجلد `TawjihiExamsJSON`.

6. **صيغة JSON للامتحان**:

## ملاحظات

- يمكن لأي شخص متمكن قراءة كود الصفحة معرفة كلمة مرور الإدارة لأنها ظاهرة في `index.html`. إذا أردت حماية أفضل يمكنك تعديل Apps Script لإجراء التحقق داخل الخادم وعدم تضمين كلمة المرور في الواجهة.
- تستخدم هذه النسخة Google Sheets كقاعدة بيانات بسيطة. إذا زاد عدد المستخدمين والامتحانات قد تحتاج إلى نقلها إلى نظام تخزين أقوى.
- التصميم مبني على Tailwind CSS وReact ويدعم الوضع الليلي والسطوع التلقائي. الواجهة متجاوبة لتعمل في المتصفح على الحواسيب والهواتف.
